# Dossier Debug - Projet Memory

Ce dossier contient les fichiers de test et de débogage pour le développement.

## Fichiers présents

- `test_users.php` : Tests manuels pour la gestion des utilisateurs
- `test_password.php` : Tests de hachage et vérification des mots de passe  
- `test_auth.html` : Interface HTML pour tester l'authentification
- `debug_headers.php` : Utilitaire pour déboguer les headers HTTP

## Usage

Ces fichiers sont destinés au développement local uniquement. 
**Ne pas inclure en production.**
